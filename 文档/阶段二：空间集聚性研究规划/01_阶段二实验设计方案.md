# 阶段二实验设计方案

## 1. 正式输入数据

阶段二主分析使用阶段一最终底板，不直接使用早期中间数据。

| 数据 | 路径 | 用途 |
|---|---|---|
| 主格网年度面板 | `阶段一：企业动态变量研究/7、最终格网标准数据/2、面板数据/master_grid_year_panel.csv` | 年度分布、Moran、LISA、Gi* |
| 主格网基础元数据 | `阶段一：企业动态变量研究/7、最终格网标准数据/1、基础元数据/master_grid_metadata.csv` | 格网位置、行列号、质心 |
| 企业年度点位表 | `阶段一：企业动态变量研究/8、最终点位数据/1、主格网对齐点位表/firm_year_points_2015_2024.csv` | 点位图、核密度 |
| 完整矩形格网面板 | `阶段一：企业动态变量研究/9、最终DL格网标准数据/2、面板数据/master_full_grid_year_panel.csv` | 时空矩阵、深度学习探索备用 |

## 2. 主变量

| 字段 | 含义 | 阶段二用途 |
|---|---|---|
| `active_cnt` | 年度活跃高新企业数 | 主线变量 |
| `entry_cnt` | 进入代理量 | 新增增长空间 |
| `renew_cnt` | 续认代理量 | 资格维持能力 |
| `exit_next_cnt` | 下一年退出代理量 | 退出风险空间 |
| `capital_sum` | 注册资本总量 | 资本强度集聚 |
| `n_size_*` | 企业规模结构 | 小微/中大型差异 |
| `n_cat_*` | 技术门类结构 | 技术方向差异 |

## 3. 实验序列

### 实验 1：年度描述统计

目的：形成阶段二的数量背景。

输出：

- 年度活跃企业总量；
- 活跃格网数量；
- 高承载格网数量；
- 零值格网比例；
- 单格网最大值；
- 进入、续认、退出代理年度总量。

### 实验 2：年度空间分布图

目的：展示高新企业空间扩展和核心区变化。

建议正文展示年份：

```text
2015、2018、2021、2024
```

主图字段：

```text
active_cnt
```

补充图字段：

```text
entry_cnt、renew_cnt、exit_next_cnt、capital_sum
```

### 实验 3：Global Moran's I

目的：证明企业分布不是随机分布，而是存在整体空间集聚。

主分析：

```text
active_cnt：2015-2024 全年份
```

补充分析：

```text
entry_cnt、renew_cnt、exit_next_cnt、capital_sum
```

### 实验 4：LISA 局部空间自相关

目的：识别 HH、LL、HL、LH 类型。

正文建议使用代表年份：

```text
2015、2018、2021、2024
```

### 实验 5：Getis-Ord Gi* 热点分析

目的：识别显著热点和冷点，是阶段二核心地图成果。

主变量：

```text
active_cnt
```

动态变量：

```text
entry_cnt、renew_cnt、exit_next_cnt
```

### 实验 6：时空热点演化

目的：从静态热点图提升到时空演化类型识别。

输出类型包括：

- 新兴热点；
- 持续热点；
- 增强热点；
- 减弱热点；
- 历史热点；
- 无显著模式。

### 实验 7：企业结构分类比较

建议正文只选三组：

1. 小微企业 vs 中大型企业；
2. 电子信息、高技术服务、先进制造与自动化；
3. 企业数量集聚 vs 注册资本强度集聚。

## 4. 注意事项

1. 空间统计必须保留 `valid_mask=1` 的零值格网。
2. 不要用 `active_cnt > 0` 筛选后再做 Moran、LISA 或 Gi*。
3. `2015` 年 `entry_cnt` 存在左截断。
4. `2024` 年 `exit_next_cnt=0` 是右删失，不解释为真实无退出。
5. 边界格网主分析保留，稳健性检验可剔除 `boundary_mask=1`。
