## 2. 论文解读模板

### 2.1 年度分布

可以围绕以下逻辑写：

```text
2015-2024 年广州市高新技术企业活跃数量持续增加，活跃格网数量同步扩展，说明高新技术企业空间分布由少数核心区向外围区域扩散。但高承载格网和高值区仍集中分布，表明扩散过程中仍保留明显核心集聚特征。
```

### 2.2 Global Moran's I

```text
Global Moran's I 结果显示，研究期内 active_cnt 在 500m 格网尺度下呈显著正向空间自相关，说明高新技术企业并非随机分布，而是在空间上表现出显著集聚。
```

### 2.3 LISA

```text
LISA 结果进一步揭示了局部集聚类型。HH 类型可解释为成熟高新企业集聚核心，LL 类型反映长期低活跃区域，HL 类型表示局部高值孤岛，LH 类型则可能对应核心区周边尚未充分承载的低值洼地。
```

### 2.4 Gi* 热点

```text
Getis-Ord Gi* 热点分析显示，高新技术企业热点区具有连续空间结构。活跃存量热点代表成熟集聚区，进入热点代表新增增长空间，续认热点代表资格维持能力较强区域，退出代理热点则提示潜在风险区域。
```

### 2.5 时空热点演化

```text
时空热点演化结果可将热点进一步区分为持续热点、新兴热点、增强热点和衰退热点。持续热点对应稳定创新承载区，新兴热点反映未来增长空间，衰退或历史热点提示部分区域集聚优势可能减弱。
```

## 3. 预期研究判断

基于现有数据规模，阶段二大概率会得到：

1. `active_cnt` 显著正向空间自相关；
2. 成熟热点集中于核心创新与产业平台区域；
3. `entry_cnt` 热点比 `active_cnt` 更能反映新增长空间；
4. `renew_cnt` 热点更接近成熟创新生态区；
5. `exit_next_cnt` 需要谨慎解释，尤其不能使用 2024 年退出代理做真实退出判断。
