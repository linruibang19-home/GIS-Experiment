# 阶段二：空间集聚性研究规划导航

## 1. 阶段定位

阶段二对应论文第四章，目标是回答：

1. 广州市高新技术企业是否存在显著空间集聚；
2. 2015-2024 年集聚热点在哪里，是否扩展、强化或衰退；
3. 活跃存量、进入、续认和退出代理的空间格局是否不同；
4. 企业规模、技术门类和资本强度是否呈现差异化集聚。

1. 年度描述统计表；
2. 2015、2018、2021、2024 年活跃存量空间分布图；
3. 2015-2024 年 Global Moran's I 表和趋势图；
