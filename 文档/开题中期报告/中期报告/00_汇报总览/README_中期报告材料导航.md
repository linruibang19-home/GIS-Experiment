# 中期报告材料导航

本目录用于支撑《广州市高新技术企业空间集聚的多尺度影响因素与时空预测研究》中期答辩。当前版本为增强版阶段一汇报材料，重点展示 7、8、9 三类最终 CSV 数据底板的处理成果、统计特征、企业属性结构、格网承载特征和 DL 矩阵预测底板。

当前图件不是后续 ArcGIS Pro 空间集聚实验、影响因素实验或预测模型实验的最终结果图，而是阶段一数据处理成果展示图。

## 目录结构

```text
中期报告/
├─ 00_汇报总览/
├─ 01_报告文本/
├─ 02_统计表格/
├─ 03_汇报图件/
├─ 04_答辩讲稿/
└─ 05_脚本/
```

## 增强版图件体系

```text
03_汇报图件/
├─ 01_数据底板结构          2张
├─ 02_年度增长与动态变量    4张
├─ 03_格网扩展与承载        3张
├─ 04_企业属性结构          5张
├─ 05_企业生命周期          3张
├─ 06_格网结构特征          2张
├─ 07_DL矩阵预测底板        3张
└─ 08_数据一致性校验        2张
```

每张图均输出 PNG、PDF、SVG 三种格式。其中 PNG 为 600ppi，PDF/SVG 可作为 PPT 或论文排版中的高清矢量版本。

## 当前版本内容

- `01_报告文本/01_阶段一数据处理成果中期汇报.md`：增强版阶段一成果汇报文字稿。
- `01_报告文本/03_汇报图件说明.md`：24 张图件的用途说明。
- `02_统计表格/`：增强版年度汇总、门类结构、规模结构、生命周期、格网稳定性、集中度和校验表。
- `03_汇报图件/`：24 张增强版图件，每张包含 PNG/PDF/SVG。
- `04_答辩讲稿/`：PPT 页码设计、讲述口径和答辩问答。
- `05_脚本/build_stage1_csv_figures.py`：读取阶段一 7、8、9 CSV 并生成全部图表的脚本。

## 核心校验结论

- 点位表年度记录数等于有效格网面板 `active_cnt` 年度总和，最大差值为 0。
- 完整格网面板 `valid_mask` 年度总数等于有效格网面板 `valid_mask` 年度总数，最大差值为 0。
- 完整格网年度单元数稳定。
- 年份范围为 2015-2024。

因此，阶段一数据处理成果已经可以支撑后续三大研究框架：空间集聚性研究、空间分布影响因素研究和年度时空预测研究。
