# 阶段一数据处理核心成果中期汇报版

## 一、阶段一数据处理目标

阶段一的目标是将原始高新技术企业认定名单、企业工商属性信息和企业空间点位数据，统一处理为可用于空间集聚性研究、空间分布影响因素研究和年度时空预测研究的标准化数据底板。

处理后的核心成果包括三类：

| 成果编号 | 成果名称 | 核心用途 |
|---|---|---|
| 7 | 最终格网标准数据 | 支撑空间集聚性研究和影响因素研究 |
| 8 | 最终点位数据 | 支撑企业年度点位、属性结构和企业空间稳定性分析 |
| 9 | 最终 DL 格网标准数据 | 支撑年度时空预测和深度学习建模 |

## 二、数据获取与处理全流程

阶段一的数据处理流程可以概括为：

```text
公开年度高新技术企业认定名单
  -> 筛选并整理广州市 2013-2024 年企业名单
  -> 通过天眼查批量查询补充工商属性
  -> 合并企业认定信息、工商属性和空间点位
  -> 清洗企业名称、统一社会信用代码、认定年份、地址、坐标、注册资本、企业规模、技术门类
  -> 构建企业认定事件表和企业主表
  -> 根据高新资格有效期展开企业年度活跃状态
  -> 构建活跃、新增、持续、续认和退出代理等动态变量
  -> 将企业年度点位匹配至广州市 500m 主格网
  -> 按 grid_id + year 汇总格网年度变量
  -> 统一旧 clip 格网与新主格网编号体系
  -> 形成最终格网标准数据、最终点位数据和最终 DL 格网标准数据
```

本流程实现了从“企业名单数据”到“500m 格网年度面板数据”的转换，使后续研究能够在统一空间单元和统一时间口径下开展。

## 三、企业动态变量推导公式

设企业为 `i`，年份为 `t`，格网为 `g`。企业是否处于高新资格活跃状态记为 `present(i,t)`。

### 1. 企业年度活跃变量

企业在年份 `t` 处于高新资格有效状态，则：

```text
present(i,t) = 1
```

否则：

```text
present(i,t) = 0
```

格网年度活跃企业数量为：

```text
active_cnt(g,t) = sum present(i,t), i 属于格网 g
```

### 2. 新增进入变量

企业当前年活跃、上一年不活跃，则视为新增进入代理：

```text
entry_proxy(i,t) = 1, 若 present(i,t)=1 且 present(i,t-1)=0
entry_proxy(i,t) = 0, 否则
```

格网汇总为：

```text
entry_cnt(g,t) = sum entry_proxy(i,t), i 属于格网 g
```

说明：2015 年为面板起始年，`entry_cnt` 不宜完全解释为真实新增，而应解释为样本期起点进入代理。

### 3. 持续活跃变量

企业当前年活跃、上一年也活跃，则视为持续活跃：

```text
persist(i,t) = 1, 若 present(i,t)=1 且 present(i,t-1)=1
persist(i,t) = 0, 否则
```

格网汇总为：

```text
persist_cnt(g,t) = sum persist(i,t), i 属于格网 g
```

### 4. 续认变量

企业在当前年份发生高新认定事件，且此前已存在认定事件，则视为续认：

```text
renew_proxy(i,t) = 1, 若 cert_event(i,t)=1 且 存在 s<t 使 cert_event(i,s)=1
renew_proxy(i,t) = 0, 否则
```

格网汇总为：

```text
renew_cnt(g,t) = sum renew_proxy(i,t), i 属于格网 g
```

### 5. 下一年退出代理变量

企业当前年活跃、下一年不活跃，则视为下一年退出代理：

```text
exit_proxy_next_yr(i,t) = 1, 若 present(i,t)=1 且 present(i,t+1)=0
exit_proxy_next_yr(i,t) = 0, 否则
```

格网汇总为：

```text
exit_next_cnt(g,t) = sum exit_proxy_next_yr(i,t), i 属于格网 g
```

说明：2024 年缺少 2025 年完整观测，因此 `exit_next_cnt=0` 不能解释为真实无退出，而应视为右删失。

## 四、最终成果 7：格网标准数据

### 1. 数据定位

目录：

```text
阶段一：企业动态变量研究/7、最终格网标准数据
```

该成果是后续空间集聚性研究和空间分布影响因素研究的主要数据底板。其核心数据表为：

```text
master_grid_year_panel.csv
```

该表包含：

```text
29,755 个有效主格网 × 10 年 = 297,550 条记录
```

### 2. 核心字段说明

| 字段 | 含义 | 汇报中可解释为 |
|---|---|---|
| `grid_id` | 主格网标准 ID | 500m 格网唯一编号 |

| `row_id`、`col_id` | 格网行列号 | 格网空间索引 |
| `centroid_x`、`centroid_y` | 格网质心坐标 | 空间定位 |
| `cover_ratio` | 广州市范围覆盖比例 | 边界格网覆盖程度 |
| `boundary_mask` | 边界格网标记 | 边界稳健性检验 |
| `valid_mask` | 有效格网标记 | 是否属于广州市有效研究范围 |
| `active_cnt` | 活跃企业数 | 空间集聚主变量 |
| `entry_cnt` | 新增进入代理数 | 新增增长空间 |
| `persist_cnt` | 持续活跃企业数 | 企业空间稳定性 |

## 六、最终成果 9：最终 DL 格网标准数据

### 1. 数据定位

目录：

```text
阶段一：企业动态变量研究/9、最终DL格网标准数据
```

核心表：

```text
master_full_grid_year_panel.csv
```

该表包含：

```text
68,930 个完整矩形格网 × 10 年 = 689,300 条记录
```

该成果与 `7、最终格网标准数据` 的区别在于：  
`7` 只保留广州市内有效格网，`9` 保留完整矩形格网，包括市外补齐格网，用于构建规则矩阵和深度学习输入。

### 2. 核心字段说明

| 字段 | 含义 | 汇报中可解释为 |

| `year` | 年份 | 年度时间维度 |
| `row_id`、`col_id` | 行列号 | 二维矩阵索引 |
| `centroid_x`、`centroid_y` | 质心坐标 | 空间定位 |
| `master_area` | 标准格网面积 | 完整 500m 格网面积 |
| `cover_ratio` | 广州市覆盖比例 | 市内/边界/市外识别 |
| `boundary_mask` | 边界格网标记 | 边界控制 |
| `valid_mask` | 有效格网标记 | 1 为广州市内有效格网，0 为市外补齐格网 |
| `active_cnt` | 活跃企业数 | 预测目标或节点特征 |

该成果可以组织为：


## 九、中期汇报可用表述

阶段一已经完成从原始企业名单到 500m 格网年度面板的数据底板建设。通过整理 2013-2024 年高新技术企业认定名单、补充企业工商属性、清洗企业空间点位并展开企业年度状态，研究形成了企业年度点位表、有效格网年度面板和完整 DL 格网年度面板三类核心成果。该阶段共形成 99,498 条企业年度点位记录、297,550 条有效格网年度面板记录和 689,300 条完整 DL 格网年度面板记录，为后续空间集聚性研究、空间分布影响因素研究和年度时空预测研究提供了统一的数据基础。
