# 阶段一增强版CSV汇报图件说明

本批图件基于阶段一 7、8、9 三类最终CSV产出生成，用于中期答辩展示数据处理成果，不替代后续ArcGIS Pro空间集聚、影响因素和时空预测实验结果图。

## 输出格式

- 每张图均输出 PNG、PDF、SVG 三种格式。
- PNG 为 600ppi；PDF/SVG 为矢量格式，适合放入PPT或后续排版。

## 图件分组

### 01_数据底板结构
- `fig01_阶段一核心CSV数据底板规模`：展示三类核心CSV的数据规模。
- `fig02_阶段一CSV数据底板关系`：展示企业点位、有效格网面板、完整DL格网面板关系。

### 02_年度增长与动态变量
- `fig03_活跃企业数量与年度净增长`：展示活跃企业数量及年度净增长。
- `fig04_活跃企业同比增长率`：展示年度增长率。
- `fig05_企业动态变量年度变化`：展示新增、持续、续认定和退出代理。
- `fig06_企业动态变量年度占比结构`：展示动态变量结构比例。

### 03_格网扩展与承载
- `fig07_格网覆盖范围与承载强度变化`：展示活跃格网数量和格网企业承载强度变化。
- `fig08_高承载格网数量年度变化`：展示高承载格网数量。
- `fig09_格网企业承载集中度变化`：展示Gini、HHI和Top格网承载比例。

### 04_企业属性结构
- `fig10_技术门类年度结构变化`：展示技术门类年度占比。
- `fig11_主要技术门类年度数量变化`：展示主要技术门类数量变化。
- `fig12_企业规模年度结构变化`：展示企业规模结构。
- `fig13_技术门类与企业规模结构交叉`：展示门类与规模交叉结构。
- `fig14_注册资本年度分布_对数尺度`：展示注册资本长尾分布。

### 05_企业生命周期

### 07_DL矩阵预测底板

- `fig21_DL非零格网比例变化`：展示DL面板零膨胀程度。

### 08_数据一致性校验
- `fig23_点位与格网年度数量一致性`：展示点位与格网数量一致。
- `fig24_CSV数据一致性校验`：展示四项校验结果。

## 关键校验

- 点位表年度记录数 = 有效格网面板active_cnt年度总和：PASS，最大差值 0
- 完整格网面板valid_mask年度总数 = 有效格网面板valid_mask年度总数：PASS，最大差值 0
- 完整格网年度单元数稳定：PASS，最大差值 0
- 年份范围为2015-2024：PASS，最大差值 0

## 核心统计口径

- 年份范围：2015-2024
- 企业年度点位记录数：99,498
- 有效格网年度面板记录数：297,550
- 完整DL格网年度面板记录数：689,300
- 2015年活跃企业数：1,865
- 2024年活跃企业数：13,523
