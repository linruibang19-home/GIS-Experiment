# 数据处理步骤输入输出链

## 1. 总链条

```text
全国高新技术企业数据库
    ↓
广州市年度高企数据
    ↓
天眼查工商属性合并
    ↓
核心字段 + 高新门类 + 坐标转换年度表
    ↓
step1 合并清洗
    ↓
step2 企业认定事件表
    ↓
step3 企业主表
    ↓
step4 企业年份活跃表
    ↓
step5 状态变量与退出代理表
    ↓
step6 一致性检查
    ↓
ArcGIS 企业点位与 clip 格网空间连接
    ↓
step7-step9 clip 格网年度面板
    ↓
ArcGIS 构建 clip → master 映射
    ↓
step11 主格网标准化
    ↓
step12 企业点位主格网对齐
    ↓
step13 完整矩形格网归档
```

## 2. 前置整理阶段

| 步骤 | 输入 | 处理 | 输出 |
|---|---|---|---|
| 广州市筛选 | 全国高新技术企业数据库 | 按所属城市筛选广州 | 广州市年度高企 Excel |
| 天眼查查询文件生成 | 广州年度高企 Excel | 提取统一社会信用代码 | 批量查询 Excel |
| 工商信息采集 | 批量查询 Excel | 天眼查人工批量查询 | 工商信息结果 Excel |
| 工商属性合并 | 高企表 + 天眼查表 | 统一社会信用代码内连接 | 工商属性合并宽表 |
| 核心字段提取 | 合并宽表 | 提取核心字段、推测门类 | 年度核心字段表 |
| 坐标清洗 | 年度核心字段表 | API 补坐标、修区县、删异常 | 清洗后年度表 |
| 坐标转换 | GCJ-02 坐标 | 转 WGS84 / CGCS2000 | 28 字段年度表 |

## 3. 企业级主流程

| Step | 脚本 | 输入 | 输出 | 规模 |
|---:|---|---|---|---:|
| 1 | `step1_merge_and_clean.py` | 2013-2024 年度企业表 | `step1_merged_cleaned.csv` | 38,400 |

| 6 | `step6_consistency_check.py` | step1-step5 输出 | 一致性报告 | 已通过 |

## 4. 格网化与主格网统一

| 步骤 | 输入 | 处理 | 输出 |
|---|---|---|---|
| 创建格网 | 广州市行政区边界 | ArcGIS 创建 500m clip 格网 | `grid_base_clip` |
| 企业转点 | 企业年份活跃表 | 经纬度转点 | 企业年份点图层 |
| 空间连接 | 企业点 + clip 格网 | 点落入格网 | `firm_joinWith_gridID.csv` |
| 格网统计 | 企业点格网表 | 按 `grid_id + year` 聚合 | 非完整统计表 |
| 完整面板 | 所有格网 × 年份 | 缺失填 0 | `grid_year_panel_final.csv` |
| 年度拆分 | 格网年度面板 | 按年拆分 | `grid_panel_YYYY.csv` |
| 主格网映射 | clip 格网 + 完整主格网 | 空间匹配 | `clip_to_master_grid_mapping.csv` |
| 主格网标准化 | 旧面板 + 映射表 | 替换主键和坐标 | `master_grid_year_panel.csv` |
| 完整矩形面板 | 主格网元数据 + 市内面板 | 市外补齐 0 | `master_full_grid_year_panel.csv` |
