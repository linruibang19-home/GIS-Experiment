# 阶段一：实验数据处理记录导航

## 1. 文档目的

本目录用于记录论文《广州市高新技术企业空间集聚的多尺度影响因素与时空预测研究》第一阶段已经完成的数据处理工作，防止处理过程、字段口径、关键脚本、成果目录和注意事项丢失。

本目录只记录已经完成的工作和当前可确认的处理逻辑，不直接修改任何数据文件。

## 2. 阶段一总体定位

阶段一的核心任务是：

> 将原始逐年高新技术企业认定名单，经过工商属性补充、空间坐标清洗、企业动态变量构建、500m 格网化和主格网标准化，最终形成可支撑后续三类实验的统一数据底板。

后续三类实验包括：

1. 空间集聚性研究；
2. 多尺度影响因素研究；
3. 年度时空预测研究。
