# 关键字段与变量口径说明

## 1. 企业唯一标识

| 字段 | 含义 |
|---|---|
| `firm_id` | 企业唯一 ID，使用统一社会信用代码 |
| `credit_code` | 统一社会信用代码，通常与 `firm_id` 相同 |
| `company_name` | 企业名称 |

说明：

- 同一企业跨年份可能多次出现，这是正常的认定事件结构；
- 同一企业同一年重复记录在认定事件表中已按 `(firm_id, cert_year)` 去重。

## 2. 时间字段

| 字段 | 含义 |
|---|---|
| `source_year` | 原始年度文件年份 |
| `cert_year` | 解析得到的认定年份 |
| `cert_valid_from` | 证书有效期起始日期 |
| `cert_valid_to` | 证书有效期结束日期 |
| `year` | 企业活跃年份或格网面板年份 |
| `cert_year_source` | 某条活跃记录由哪一年认定事件支撑 |

研究期：

```text
2015-2024
```

原因：

- 2015 年活跃存量可由 2013、2014、2015 三年认定事件完整重建；
- 2013、2014 年无法完整作为研究期存量年份；
- 2024 年是当前右端年份，退出变量存在右删失。

## 3. 企业活跃状态变量

| 字段 | 定义 |
|---|---|
| `present` | 企业在某年处于高新技术企业资格活跃状态 |
| `entry_proxy` | 本年活跃且上年不活跃 |
| `persist` | 本年活跃且上年也活跃 |
| `renew_proxy` | 本年发生认定事件且上年已活跃 |
| `exit_proxy_next_yr` | 本年活跃但下一年不活跃 |

### 3.1 活跃状态

每条认定事件按 3 年有效期展开：

```text
cert_year = y
present year = y, y+1, y+2
```

### 3.2 进入代理量

企业在年份 `t` 满足：

```text
present(i,t)=1 且 present(i,t-1)=0
```

则：

```text
entry_proxy = 1
```

注意：

- 2015 年是研究期起点，该年进入量包含左截断影响；
- 论文中应谨慎将 2015 年进入量解释为真实新增。

### 3.3 持续活跃量

企业在年份 `t` 满足：

```text
present(i,t)=1 且 present(i,t-1)=1
```

则：

```text
persist = 1
```

### 3.4 续认代理量

企业在年份 `t` 满足：

```text
当年有认定事件，且上一年处于活跃状态
```

则：

```text
renew_proxy = 1
```

续认代理量用于衡量企业资格维持和再次认定能力。

### 3.5 退出代理量

企业在年份 `t` 满足：

```text
present(i,t)=1 且 present(i,t+1)=0
```

则在年份 `t` 的记录中：

```text
exit_proxy_next_yr = 1
```

注意：

- 这里的退出是高新技术企业资格退出代理，不等同于企业注销；
- 2024 年没有 2025 年观测，不能将 `exit_proxy_next_yr=0` 解释为真实没有退出。

## 4. 企业属性变量

| 字段 | 含义 |
|---|---|
| `capital_wan` | 注册资本，单位万元 |
| `size_class` | 企业规模，取值为大型、中型、小型、微型、未知 |
| `hi_category` | 高新技术企业八大门类 |
| `enterprise_status` | 企业工商状态 |
| `district` | 企业属性表中的所属区县 |

注意：

- 注册资本、规模、工商状态含天眼查查询时点影响；
- 天眼查数据反映查询时点的工商属性，不一定完全代表历史年份真实状态；
- 在论文中应说明这些字段作为企业结构属性使用。

## 5. 坐标字段

| 字段 | 坐标系 | 用途 |
|---|---|---|
| `lon_gcj02` | GCJ-02 | 原始或高德坐标 |
| `lat_gcj02` | GCJ-02 | 原始或高德坐标 |
| `lon` | WGS84 | 通用经纬度 |
| `lat` | WGS84 | 通用经纬度 |
| `lon_cgcs` | CGCS2000 geographic，EPSG:4490 | ArcGIS 建点 |
| `lat_cgcs` | CGCS2000 geographic，EPSG:4490 | ArcGIS 建点 |

空间图层投影：

```text
EPSG:4547
CGCS2000 3-degree Gauss-Kruger CM 114E
```

## 6. 格网标识字段

| 字段 | 含义 |
|---|---|
| `grid_id` | 主格网标准 ID，格式为 `RxxxCxxx` |
| `row_id` | 主格网行号 |
| `col_id` | 主格网列号 |
| `centroid_x` | 主格网标准质心 X |
| `centroid_y` | 主格网标准质心 Y |
| `master_area` | 完整主格网面积，理论上约 250,000 平方米 |

主格网 ID 示例：

```text
R079C109
```

含义：

```text
第 79 行，第 109 列
```

## 7. 边界和掩膜字段

| 字段 | 含义 |
|---|---|
| `cover_ratio` | clip 格网面积 / 完整主格网面积 |
| `boundary_mask` | 是否边界格网，1 是，0 否 |
| `valid_mask` | 是否广州市内有效格网，1 是，0 否 |

说明：

- `valid_mask=1` 用于空间统计和影响因素分析；
- `valid_mask=0` 是完整矩形中用于补齐的市外格网；
- `boundary_mask=1` 的格网应在稳健性检验中单独处理；
- 当前有效边界格网数为 1,815。

## 8. 格网年度统计变量

| 字段 | 含义 |
|---|---|
| `active_cnt` | 当年活跃企业数量 |
| `entry_cnt` | 当年进入代理数量 |
| `persist_cnt` | 当年持续活跃数量 |
| `renew_cnt` | 当年续认代理数量 |
| `exit_next_cnt` | 当年活跃但下一年退出的代理数量 |

关键关系：

```text
active_cnt = entry_cnt + persist_cnt
```

当前主面板中该关系全部成立。

## 9. 注册资本统计变量

| 字段 | 含义 |
|---|---|
| `capital_mean` | 格网年度企业注册资本均值 |
| `capital_median` | 格网年度企业注册资本中位数 |
| `capital_sum` | 格网年度企业注册资本总和 |

## 10. 企业规模分布变量

| 字段 | 含义 |
|---|---|
| `n_size_微型` | 微型企业数量 |
| `n_size_小型` | 小型企业数量 |
| `n_size_中型` | 中型企业数量 |
| `n_size_大型` | 大型企业数量 |
| `n_size_未知` | 未知规模企业数量 |

当前主面板中：

```text
active_cnt = n_size_微型 + n_size_小型 + n_size_中型 + n_size_大型 + n_size_未知
```

全部成立。
